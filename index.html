Títulos: Tags <h1> até <h6>

As tags <h1> até <h6> em HTML são utilizadas para criar títulos e subtítulos em uma página web, com <h1> representando o título mais importante e <h6> o menos importante.

Também acontece uma mudança no tamanho dos títulos sendo que a tag <h1> tem o maior tamanho e a tag <h6> tem o menor tamanho. Mas todas ficam destacadas em negrito.


Exemplo: "<h1> Meu Titulo Principal </h1>"

Parágrafos: Tag <p>

A tag <p> em HTML é usada para definir parágrafos de texto. Essa tag é essencial para organizar e estruturar o conteúdo de uma página web em blocos de texto separados.

Exemplo: "<p> Esse é um parágrafo </p>"

Destacar texto: Tag <mark>

A tag <mark> em HTML é usada para destacar partes do texto que são relevantes ou foram marcadas de alguma forma.

Ela é comumente utilizada para evidenciar palavras-chave ou termos de busca em resultados de pesquisa.

Exemplo: <p> Este é um parágrafo de exemplo com uma palavra marcada: <mark> destaque <mark>. </p>


Neste exemplo, a palavra "destaque" será destacada visualmente na página devido à utilização da tag <mark>.

Representar o texto com uma fonte menor: Tag <small>

A tag <small> é usada em HTML para indicar que o texto contido dentro dela deve ser representado em uma fonte menor do que o texto circundante.


Exemplo: <p>Este é um parágrafo com uma nota <small>sobre um detalhe adicional</small>.</p>

Neste exemplo, a frase "sobre um detalhe adicional" será exibida em uma fonte menor devido ao uso da tag <small>.

Representar o texto em italico: Tag <i>

A tag <i> em HTML é usada para representar texto em itálico. Antigamente, ela era comumente usada para destacar texto em itálico quando a semântica específica do texto não era crucial, mas com a adoção mais ampla do HTML5 e das melhores práticas de semântica, o uso de <i> para ênfase ou itálico é considerado menos apropriado.

Em vez disso, para texto em itálico, a recomendação é usar a tag <em> (ênfase), que carrega uma semântica mais clara. A tag <em> é usada para indicar ênfase no texto, que pode ser representado visualmente como itálico pelos navegadores padrão.


Exemplo com <em>: <p>Este é um exemplo de texto com <em>ênfase</em>.</p>

Exemplo com <i>: <p>Este é um exemplo de texto <i>itálico</i>.</p>

Sublinhar o texto: Tag <u>

A tag <u> em HTML é usada para representar texto sublinhado. No entanto, seu uso é desencorajado em HTML5, pois o sublinhado costuma ser associado a links e pode causar confusão quando usado em outros contextos.

Em vez de usar <u> para sublinhar o texto, é recomendável utilizar CSS para aplicar estilos, como a propriedade text-decoration.


Exemplo: <p>Este é um <u>exemplo de texto sublinhado</u>.</p>

Neste exemplo, a tag <u> é usada para sublinhar o texto "exemplo de texto sublinhado".

Enfatizar texto: Tag <strong>

A tag <strong> em HTML é usada para indicar que o texto contido deve ser representado como forte ou enfatizado.

Exemplo: <p>Este é um <strong>exemplo de texto enfatizado</strong>.</p>

Neste exemplo, a palavra "exemplo de texto enfatizado" está envolta pela tag <strong>, indicando que é de importância mais forte ou enfatizada.

Criar Lista Ordenada: Tag <ol>

A tag <ol> em HTML é usada para criar uma lista ordenada, ou seja, uma lista de itens onde cada item é sequencialmente numerado. Aqui está uma explicação mais detalhada em português:

<ol>: Esta é a tag de abertura que indica o início de uma lista ordenada.

<li>: Cada item na lista é representado pela tag <li> (abreviação de "list item" ou item de lista). Essa tag envolve cada elemento individual da lista.

</ol>: Esta é a tag de fechamento que indica o fim da lista ordenada.


Exemplo:
<ol>

<li>Item 1</li>
<li>Item 2</li>
<li>Item 3</li>
</ol>


Voltar

Criar Lista não Ordenada: Tag <ul>

A tag <ul> em HTML é usada para criar uma lista não ordenada. Ao contrário das listas ordenadas (utilizando a tag "ol"), as listas não ordenadas não têm números sequenciais, mas geralmente são representadas por marcadores, como pontos ou círculos.


Exemplo:
<ul>

<li>Item 1</li>
<li>Item 2</li>
<li>Item 3</li>
</ul>

Adicionar itens a uma lista: Tag <li>

A tag <li> é usada em HTML para definir um item de lista. É comumente utilizada em listas ordenadas (<ol>) ou listas não ordenadas (<ul>).


Exemplo:
<ul>

<li>Item 1</li>
<li>Item 2</li>
<li>Item 3</li>
</ul>

Criar hiperlinks: Tag <a>

A tag <a> é usada em HTML para criar hiperlinks, também conhecidos como âncoras ou links.


Exemplo: <a href="https://www.exemplo.com">Texto do Link</a>

href: é um atributo que especifica o URL para o qual o link direcionará.

Texto do Link: é o texto que será exibido como o link no navegador.

Adicionar uma linha horizontal: <hr>

A tag <hr> em HTML é utilizada para criar uma linha horizontal, muitas vezes usada para separar seções de conteúdo em uma página.

Essa tag não requer um elemento de fechamento e é uma "tag vazia", o que significa que ela não possui conteúdo entre tags de abertura e fechamento.


Exemplos:
<p>Este é um parágrafo de texto.</p>
<hr>
<p>Este é outro parágrafo de texto, separado pela linha horizontal.</p>

No exemplo acima, a tag <hr> é usada para inserir uma linha horizontal entre os dois parágrafos de texto. Quando renderizado em um navegador, isso criará uma linha visível horizontalmente na página.

Representar parte do texto subscrito: Tag <sub>

A tag <sub> em HTML é usada para representar texto subscrito. Texto subscrito é geralmente exibido abaixo da linha de base normal e é comumente utilizado para notações matemáticas, fórmulas químicas, datas e outras situações em que um texto menor precisa ser exibido abaixo do texto principal.


Exemplo: H<sub>2</sub>O

No exemplo acima, o número "2" está dentro da tag <sub>, indicando que deve ser exibido como texto subscrito. Quando renderizado em um navegador, isso resultará em "H₂O", onde o "2" está abaixo da linha de base.


Voltar

Representar parte do texto sobrescrito: Tag <sup>

A tag <sup> em HTML é usada para representar texto sobrescrito. O texto sobrescrito é exibido acima da linha de base normal e é frequentemente utilizado para notações matemáticas, exponentes, datas e outras situações em que um texto menor precisa ser exibido acima do texto principal.


Exemplo: x<sup>2</sup>

No exemplo acima, o número "2" está dentro da tag <sup>, indicando que deve ser exibido como texto sobrescrito. Quando renderizado em um navegador, isso resultará em "x²", onde o "2" está acima da linha de base.

Destacar citações: <blockquote>

A tag <blockquote>em HTML é usada para indicar um bloco de texto que é uma citação de outra fonte. É frequentemente utilizado para destacar uma citação longa ou um bloco de texto que foi reproduzido de outra fonte.


Exemplo:
<blockquote>

<p>Esta é uma citação longa ou um bloco de texto de outra fonte.</p>
<blockquote>

Definir caractesrísticas de fontes: Tag <font>

A tag <font> era usada anteriormente em HTML para definir as características de fonte do texto, como tamanho, cor e tipo de fonte.

No entanto, o uso da tag <font> é considerado obsoleto em HTML5, e é recomendado utilizar Cascading Style Sheets (CSS) para estilizar o texto.


Exemplo: <font size="4" color="red" face="Arial">Este é um texto estilizado com a tag font.</font>

Representar texto deletado ou removido: Tag <del>

A tag <del> em HTML é usada para representar o texto que foi deletado ou removido de um documento.

Quando o texto é envolvido pela tag <del>, os navegadores geralmente aplicam uma linha através do texto para indicar que foi deletado.


Exemplo:
<p>Este é um parágrafo de texto original.</p>
<del>Este texto foi deletado.</del>

Neste exemplo, a tag <del> envolve o texto "Este texto foi deletado", indicando que esse conteúdo foi removido. Quando renderizado em um navegador, o texto deletado geralmente aparecerá com uma linha através dele.

Representar uma abreviação: Tag <abbr>

A tag <abbr> em HTML é usada para definir uma abreviação ou uma sigla e é frequentemente utilizada para fornecer uma expansão do significado da abreviação quando o cursor do mouse passa sobre ela.


Exemplo: <p>Você pode aprender HTML usando a <abbr title="Hypertext Markup Language">HTML<;/abbr>.<p>
